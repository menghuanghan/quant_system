## 💾 数据存储说明

### 文件组织结构
```
data/raw/structured/
├── metadata/              # 基础元数据（10个文件）
├── market_data/           # 市场行情（7个文件夹，按股票/指数拆分）
├── fundamental/           # 公司基本面（4个文件 + 10个文件夹）
├── trading_behavior/      # 交易行为（5个文件 + 3个文件夹）
├── cross_sectional/       # 板块行业（4个文件）
├── derivatives/           # 衍生品（4个文件 + 6个文件夹）
├── index_benchmark/       # 指数基准（2个文件 + 1个文件夹）
├── macro_exogenous/       # 宏观数据（8个文件）
└── deep_risk_quality/     # 风险因子（5个文件）
```

### 数据格式
- **格式**：Parquet（Apache Parquet 列式存储）
- **压缩**：Snappy 压缩算法
- **编码**：UTF-8

### 数据加载示例
```python
import pandas as pd

# 加载单个文件
df = pd.read_parquet('data/raw/structured/metadata/stock_list_a.parquet')

# 加载按股票拆分的数据
df = pd.read_parquet('data/raw/structured/market_data/stock_daily/000001_SZ.parquet')

# 批量加载某个文件夹下所有数据
import glob
files = glob.glob('data/raw/structured/market_data/stock_daily/*.parquet')
df_list = [pd.read_parquet(f) for f in files]
df_all = pd.concat(df_list, ignore_index=True)
```
